# Sweet-potato-clock(番薯打卡)
## 简介
    2019年微信小程序应用开发赛小程序组全国二等奖
## 功能介绍：
    - 自定义打卡目标（可指定时常、周期、标签）
    - 小组功能
       - 创建小组
       - 加入小组
       - 组内排名
       - 根据打卡标签智能推荐小组
    - 统计功能
       - 每日/周/月数据统计
    - 集卡游戏
      - 根据打卡市场获得积分，随机获得掉落卡片（暂不成熟）
## 技术栈简介
    - 前端采用微信小程序原生框架
    - 后端采用Spring+SpringBoot+Mybatis
    - 数据库采用Mysql/MariaDB
